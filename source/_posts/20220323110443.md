以下是本博客对主题配置进行的修改，如果你想参照我的博客样式来修改你的博客，可以直接使用以下配置，或者去我的 [Github](https://github.com/adelasky/blog) 仓库查看博客源文件。

自定义 css 项：

```css
/* 全局行高 */
.markdown-body {
    line-height: 2 !important;
}

/* 段落样式 */
.markdown-body p {
    margin-bottom: 2em !important;
    text-align: justify;
}

/* 面板阴影 */
#board {
    box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1),0 4px 6px -2px rgba(0, 0, 0, 0.05) !important;
}

/* 首页悬浮动画效果 */
.index-card:hover .index-img img {
    transition: 0.3s;
    transform: scale(1.1);
}

.index-card:hover .index-info {
    transition: 0.3s;
    transform: translateX(20px) translateY(0);
}

.index-img img {
    transition: 0.3s;
}

.index-info {
    transition: 0.3s;
}

@media (max-width: 767px) {
    .index-card:hover .index-img img {
        transform: none !important;
    }
    .index-card:hover .index-info {
        transform: none !important;
    }
}
```
